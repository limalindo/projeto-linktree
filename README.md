# Projeto Linktree - Lima

Um linktree pessoal desenvolvido para Lucas Gomes (@limaayzz) com links para suas redes sociais e plataformas.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Font Awesome (ícones)
- Design responsivo

## 📱 Funcionalidades

- Design moderno com animações
- Layout responsivo para mobile e desktop
- Links diretos para redes sociais
- Animações suaves e efeitos visuais
- Gradiente animado no background
